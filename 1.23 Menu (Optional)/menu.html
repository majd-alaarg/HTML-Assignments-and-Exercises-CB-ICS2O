1.08 Creating Links Using the link tag to add links to text.
<a href="site url in quotes">text to be linked/>
Absolute paths take the user to an external defined website http://www.mysite.com
Relative paths take the user to a file on the same page index.html or image.png
1.09 Relative vs Absolute Paths Difference between relative and absolute paths
Absolute paths, when clicked, take the user to an external web link where as a relative path is linked to a file within the same host folder as the index or original page.
1.10 Link in new window Just add attribute target="_blank"
<a href="website.com" target="_blank"><Go to CB's homepage</a>
1.11 Anchoring same page Anchors allow users to scroll to a specific HTML element on a page. These are also known as bookmarks. To do that you have assign an ID to the element , and then, set the URL of the anchor to that ID prepended by a "#" character, like this:
<a href="blah">blah</a>
<a href="#blah">go to blah</a>

1.12 Lists
Ordered list:
<ol></ol>
Use the type attribute with in the tag
type="1" Numbered list (default)
Type="A" Arranged with capital uppercase letters
Type="a" Arranged with capital lowercase letters
Type="I" Arranged with capital uppercase Roman Numerals
Type="i" Arranged with capital lowercase Roman Numerals

Unorder list:
<ul></ul>
Use the style="list-style-type:" attribute with in the tag
style="list-style-type:disc" Bullet point (default)
style="list-style-type:circle" Empty circle point
style="list-style-type:square" Arranged with square points
style="list-style-type:none" Items not marked
